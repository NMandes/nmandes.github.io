NATHANIEL MANDES

What is Deep Packet Inspection (DPI)?

  • Form of network packet filtering
  • Operates up to OSI layer 7
  • Used most notably at network demarcation points

When did DPI get introduced?

  • ARPAnet
  • OSI Layers

Types of Packet Inspections

  • Shallow | Packet Header Inspection
  • Medium | Application Proxy Inspection
  • Deep | Packet Data Inspection

Who Uses DPI?

  • Enterprise Level Companies
  • Internet Service Providers
  • Government Agencies

Leaders in DPI?

  • NetScreen Technologies
  • Cisco
  • Check Point
  • Procera

Why use DPI?

  • Network Security
  • Network Optimization & Trends
  • Law Enforcement Regulation

What does DPI prevent?

  • Worms
  • Buffer Overflow Attacks
  • Denial of Service (DoS) Attacks

How is DPI deployed?

  • Software Defined Networks using signature-matching
  • Intrusion Detection | Intrusion Prevention Firewalls
  • Port Mirroring
  • Proxy Servers with Session Control

Vulnerabilities with DPI

  • VoIP Protocols
  • Software Injections
  • Cross-Site Scripting Attacks
  • Encryption

Strategies deployed with DPI

  • Network-Address Translation (NAT)
  • Virtual Private Networking (VPN)
  • Traffic Shaping
  • Denial of Service
  • Rerouting
  • Data Mining

Controversies

  • Net Neutrality
  • Eavesdropping
  • Censorship
  • Privacy
  • Encryption
